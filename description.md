# Aura: Multi-agent desktop orchestrator powered by NVIDIA NIM

Aura is a sophisticated desktop assistant and automation framework that leverages a multi-agent system to provide seamless interaction between the user and their digital workspace. Built with a modular "Brain" (orchestration) and "Body" (tools) architecture, Aura can perform complex tasks ranging from deep research and content creation to system monitoring and UI automation.

## Key Highlights
- **Intelligent Orchestration**: Powered by LangGraph and NVIDIA NIM/OpenRouter.
- **Desktop Mastery**: Full control over UI, keyboard, and clipboard.
- **Extensible Architecture**: Support for Model Context Protocol (MCP) and custom python plugins.
- **Unified Interface**: Real-time interaction via a modern Tauri-based desktop app and a rich CLI.
